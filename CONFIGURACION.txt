CONFIGURACIÓN EN INTELLIJ IDEA
------------------------------

1. Abrir IntelliJ IDEA

2. File > Open > Seleccionar la carpeta del proyecto

3. Configurar Tomcat:
   - Run > Edit Configurations
   - Click en "+" > Tomcat Server > Local
   - Name: Tomcat 10.1.46
   - Application Server: Click "Configure" y seleccionar la ruta de instalación de Tomcat
   - En la pestaña "Deployment":
     - Click "+" > Artifact > biblioteca_apec:war exploded
     - Application context: /biblioteca_apec

4. Configurar JDK:
   - File > Project Structure > Project
   - SDK: Seleccionar OpenJDK 23
   - Language level: 23

5. Configurar Maven:
   - View > Tool Windows > Maven
   - Click en el botón "Reload All Maven Projects"

CÓMO EJECUTAR EL PROYECTO
-------------------------

1. Asegurarse de que Tomcat 10.1.46 esté instalado

2. En IntelliJ IDEA:
   - Click en el botón "Run" (triángulo verde)
   - O presionar Shift + F10

3. El navegador se abrirá automáticamente en:

4. Desde allí podrá acceder a todos los módulos

PRÓXIMOS PASOS PARA COMPLETAR EL SISTEMA
----------------------------------------

Para completar todos los requerimientos funcionales, se deben crear:

1. Módulos adicionales (siguiendo el mismo patrón):
   - Editoras
   - Idiomas
   - Usuarios
   - Empleados
   - Libros (con relaciones a otras tablas)
   - Préstamos (proceso completo de préstamo y devolución)

2. Funcionalidades avanzadas:
   - Consultas por criterios múltiples
   - Reportes (por tipo, idioma, fechas)
   - Validaciones más robustas
   - Manejo de errores personalizado

3. Mejoras de UI:
   - Paginación de tablas
   - Búsqueda y filtros
   - Mensajes de confirmación (toast notifications)

NOTAS IMPORTANTES
-----------------

- El proyecto usa @WebServlet en lugar de configuración en web.xml para los servlets
- Todas las tablas tienen Row Level Security (RLS) habilitado en Supabase
- El código sigue el patrón MVC básico
- Se usa JDBC puro sin frameworks ORM
- Las conexiones a BD se cierran automáticamente con try-with-resources
